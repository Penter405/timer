# Rubik's Cube Timer 🧊⏱️

一個精確、現代化且支援雲端同步的魔方計時器。

## 🔗 網站連結
[**點擊這裡開始計時**](https://penter405.github.io/timer/)

## ✨ 功能特色
- **精確計時**：支援毫秒級精確度，符合 WCA 操作邏輯（長按啟動）
- **雲端同步**：使用 Google 帳號登入，成績自動保存至雲端資料庫
- **全球排行榜**：即時查看所有玩家的成績排名
- **多頁面切換**：設定頁面、排行榜頁面與更多選單 (SPA 架構)
- **隱私保護**：切換帳號時自動清除個人資料
- **自訂暱稱**：支援 Discord 風格的唯一 ID 系統 (例如 `Penter#1`)
- **響應式設計**：手機、平板、電腦皆完美支援
- **個人化按鈕**：可自訂用戶按鈕的背景和文字顏色

## 📖 使用指南

### 1. 基礎操作
- **開始/停止**：
  - **電腦**：按住 `空白鍵` 等待綠燈亮起，放開開始；按任意鍵停止
  - **手機**：按住螢幕計時區域，等待綠燈，放開開始；觸碰螢幕停止
- **生成新打亂**：按 `S` 鍵或點擊打亂公式區域
- **檢視功能**：按 `I` 鍵開啟 WCA 檢視模式 (15秒倒數)

### 2. 帳號與排行榜
1. 點擊右上角的 **Google 登入** 按鈕
2. 登入成功後，四邊形按鈕會顯示你的暱稱
3. 前往 **「更多」→「用戶設定」** 頁面設定暱稱
4. 系統會自動分配唯一 ID (例如 `SpeedCuber#1`)

### 3. 成績確認（Time Check Window）
每次計時結束後，會彈出確認視窗：
- **OK (O)**：確認成績並儲存
- **+2 (2)**：加兩秒罰時後儲存
- **DNF (D/Esc)**：成績無效，不儲存

> 💡 **提示**：括號內為鍵盤快捷鍵

### 4. 全螢幕模式（Full Mode）
按下「開始檢查」(I) 後進入全螢幕模式：
- 整個螢幕（除了按鈕）都可作為開始/停止按鈕
- 期間禁止縮放和滾動
- 計時結束後自動退出全螢幕模式
